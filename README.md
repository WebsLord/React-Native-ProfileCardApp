# React Native Profile Card App 

This project is a React Native application built for the Mobile Programming . It demonstrates advanced styling techniques, including custom fonts, theming, cross-platform shadows, and responsive layouts.

Bu proje, Mobil Programlama için geliştirilmiş bir React Native uygulamasıdır. Özel fontlar, tema yönetimi, platformlar arası gölgeler ve responsive düzenler gibi gelişmiş stil tekniklerini içerir.


## Features / Özellikler

* **Custom Fonts / Özel Fontlar:** Integration of the Montserrat font family. (Montserrat font ailesi entegrasyonu.)
* **Theme System / Tema Sistemi:** Light and Dark mode support with a custom theme provider structure. (Özel tema yapısı ile Açık ve Koyu mod desteği.)
* **Shadows / Gölgeler:** Platform-specific shadow implementation (shadows for iOS, elevation for Android). (Platforma özel gölge uygulaması.)
* **Responsiveness / Uyumluluk:** Layout adapts to different screen sizes using `useWindowDimensions`. (Düzen, `useWindowDimensions` kullanılarak farklı ekran boyutlarına uyum sağlar.)
* **Interactivity / Etkileşim:** Interactive theme toggle and like button with press feedback. (Basma geri bildirimine sahip etkileşimli tema değiştirme ve beğen butonu.)

## Installation / Kurulum

1.  **Clone the repository / Repoyu klonlayın:**
    ```bash
    git clone [https://github.com/WebsLord/React-Native-ProfileCardApp.git](https://github.com/WebsLord/React-Native-ProfileCardApp.git)
    ```

2.  **Install dependencies / Bağımlılıkları yükleyin:**
    ```bash
    npm install
    ```

3.  **Run the app / Uygulamayı çalıştırın:**
    ```bash
    npx expo start
    ```
